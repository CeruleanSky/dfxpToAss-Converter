# dfxpToAss-Converter

Python script to parse and convert dfxp (subtitle) to ass

## Usage

```
usage: dfxptoass.py [-h] -i FILE -o FILE

convert dfxp to ass

optional arguments:

    -h, --help show this help message and exit

    -i FILE input dfxp file

    -o FILE output ass file
```

## Requirements

* python 3+
* argparse
* timeit
* bs4
